# 🥽 Projeto Final do Metaverso Web 3.0 – Feira Virtual

## 📌 Descrição do Projeto

Este projeto consiste em uma aplicação 3D em Realidade Estendida (XR) desenvolvida para o Meta Quest, representando uma **feira simples** contendo:

- Um **comerciante (NPC)**
- Uma **bancada de vendas**
- **Maçãs expostas** para simular uma pequena feira.

O objetivo do projeto é demonstrar e aplicar os conhecimentos vistos na trilha de Metaverso do programa Web 3.0 

---

## 🎯 Objetivo

Criar um ambiente virtual simples que represente uma situação cotidiana (uma feira), permitindo:

- Visualização do cenário em 3D  
- Presença de um personagem (comerciante)  
- Representação de objetos comerciais (bancada e maçãs)  
- Experiência imersiva em realidade virtual  

---

## 🛠️ Tecnologias Utilizadas

- Unity (com suporte a XR)
- SDK do Meta Quest
- Linguagem C#
- Modelos 3D básicos
- Plataforma Linux (Ubuntu)

---

## 🧩 Estrutura do Cenário

O ambiente é composto por:

- **Terreno/base da feira**
- **Bancada do comerciante**
- **Maçãs posicionadas sobre a bancada**
- **Personagem representando o comerciante**

O foco do projeto não é a complexidade gráfica, mas sim a construção funcional de um cenário XR simples.

---

## ⚠️ Dificuldades Encontradas

Durante o desenvolvimento do projeto, diversas problemáticas foram enfrentadas:

### 🐧 Sistema Operacional (Ubuntu)
O desenvolvimento foi realizado no Ubuntu, o que trouxe dificuldades de compatibilidade com ferramentas e bibliotecas do SDK XR e do Meta Quest, principalmente em relação à configuração do ambiente.

### 💻 Limitações de Hardware
O computador utilizado possui **baixo desempenho**, o que impactou:

- O tempo de compilação  
- A execução do projeto  
- A renderização do ambiente 3D  

Isso dificultou testes frequentes e ajustes visuais no cenário.

### 🎮 Simulador Não Funcional
Não foi possível utilizar corretamente o simulador/emulador do Meta Quest, o que impossibilitou:

- Testar a aplicação diretamente no ambiente virtual
- Verificar corretamente interações em XR
- Simular a experiência do usuário final

Os testes ficaram limitados ao modo de visualização da Unity, sem imersão real.

---

## 📚 Aprendizados

Apesar das dificuldades, o projeto proporcionou aprendizado em:

- Configuração de projetos XR no Unity  
- Organização de cena 3D  
- Uso de objetos e personagens em ambiente virtual  
- Identificação de limitações de hardware e sistema operacional  
- Importância de testes em dispositivos reais
